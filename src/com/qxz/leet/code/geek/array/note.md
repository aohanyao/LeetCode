##### Array
数组的 insert、Delete 时间复杂度为 O(n)，insert一个元素需要将
index位置以后的元素统统往后移动+1的位置，删除同样的道理，需要将index位
置以后的元素往前移动-1的位置。扩容为50%，一次性扩容1/2。

##### Link Array
链表，双向链表和单向链表。
单向链表：next
双向链表：next/previous
insert、Delete都只需要操作指针指向即可

##### skip Array
跳表，每二分之一创建索引
